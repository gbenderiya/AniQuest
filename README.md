# Get start
1. Install [UV](https://github.com/astral-sh/uv)
2. Create virtual environment
    ```shell
    $ uv venv
    ```
3. Sync environment
    ```shell
    $ uv sync
    ```
4. Add dependencies
   ```shell
   $ uv add "[package]"
   ```
5. Remove dependencies
   ```shell
   $ uv remove "[package]"
   ```
6. Run application
   ```shell
   $ uv run app.pyu
   ```
